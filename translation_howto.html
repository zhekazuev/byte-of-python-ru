Приложение: Инструкция по переводу

  1. Полный исходный текст книги доступен в Git-репозитории https://github.com/swaroopch/byte-of-python.
  2. Пожалуйста, Создайте форк репозитория.
  3. Затем скачайте репозиторий на свой компьютер. Для этого нужно быть знакомым с Git.
  4. Прочитайте документацию Honkit, особенно раздел Markdown section.
  5. Отредактируйте файлы .md на своём родном языке.
  6. Затем следуйте указаниям в INSTALL.md о том, как сгенерировать веб-сайт, PDF, EPUB.
